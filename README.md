# Snehasis Chakraborty | Personal Portfolio

A sleek, modern, and responsive personal portfolio website built to showcase my skills, projects, and academic journey as a Computer Science and Engineering student at Haldia Institute of Technology.

![Portfolio Preview](./frontend/profile.jpg)

## 🌟 Features

- **Dark Theme UI:** Modern, premium dark mode aesthetic with glassmorphism effects.
- **Responsive Design:** Fully responsive layout that looks great on mobile, tablet, and desktop screens.
- **Smooth Animations:** Integrated scroll reveals and typewriter effects written in pure JavaScript.
- **Organized Structure:** Code is kept clean, basic, and well-organized into a dedicated frontend folder.

## 📁 Repository Structure

```
Portfolio/
├── frontend/             # The main website files
│   ├── index.html        # Main HTML structure
│   ├── style.css         # Styling, CSS variables, and responsive media queries
│   ├── script.js         # Basic Javascript for animations and interactivity
│   └── profile.jpg       # Profile picture used on the website
└── README.md             # This documentation file
```

## 🛠️ Technologies Used

- **HTML5:** Semantic structuring and layout
- **CSS3:** Flexbox, Grid, CSS Variables, and custom animations
- **Vanilla JavaScript:** DOM manipulation, scroll event listeners, and typing effects
- **FontAwesome:** Scalable vector icons
- **Google Fonts:** Custom typography (Inter and JetBrains Mono)

## 🚀 How to Run Locally

You don't need any complex build tools or servers to run this project! 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/csnehasis2-eng/Portfolio.git
   ```
2. **Navigate to the frontend directory:**
   ```bash
   cd Portfolio/frontend
   ```
3. **Open the file:**
   Simply double-click on `index.html` to open it in your default web browser, or use an extension like VS Code Live Server for hot reloading.
